# Caesar Cipher

A simple Python program that encrypts and decrypts text using the Caesar Cipher algorithm.

## 📌 About the Project
The Caesar Cipher is one of the oldest encryption techniques. It works by shifting each letter in the message by a fixed number of positions in the alphabet.

For example, with a shift of 3:
- A → D
- B → E
- C → F

This project allows users to:
- Encrypt a message
- Decrypt a message
- Choose any shift value

## 🚀 Features
- Encrypt plain text
- Decrypt encrypted text
- Custom shift values
- Handles uppercase and lowercase letters

## 🛠️ Technologies Used
